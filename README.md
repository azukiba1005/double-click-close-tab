# Double Click Closes Tab (タブ名変更機能付き)

Google Chromeのタブを簡単・快速に操作するための、軽量で機能豊富なManifest V3対応拡張機能です。

## 🌟 主な機能

### 1. ページダブルクリックでタブを閉じる
* **ダブルクリッククローズ**: Webページ上の空いているエリア（背景など）をダブルクリックするだけで、そのタブを即座に閉じることができます。
* **誤動作防止の安全設計**:
  * リンク、ボタン、入力フォーム、動画プレイヤーなどのインタラクティブな要素の上でのダブルクリックでは**動作しません**。
  * テキストを選択中（ダブルクリックで単語を選択した時など）は**動作しません**。
  * CSSでカーソルが `pointer`（手の形）になっている要素上では**動作しません**。
* **修飾キー設定**: 誤動作を完全に防ぐために、「`Shift` / `Option(Alt)` / `Control(Command)` を押しながらダブルクリックした時のみ閉じる」ようにカスタマイズ可能です。

### 2. 拡張機能アイコンのワンクリッククローズ
* ツールバーにある拡張機能のアイコンをクリックするだけで、現在のアクティブなタブを即座に閉じることができます（設定でオン/オフ切り替え可能）。

### 3. タブ名の変更と固定 (永続化)
* **Option + 1** (Windowsでは **Alt + 1**) を押すことで、スタイリッシュな入力ダイアログが表示され、現在のタブにカスタム名をつけることができます。
* **遷移後も名前を維持**: 同じタブ内で別のページに移動（リンククリックやURL直接入力）しても、変更した名前が維持され続けます。
* **自動書き換え防止**: YouTubeの動画切り替えや、通知バッジなどによってWebページ側からタイトルが動的に書き換えられても、設定したカスタム名を強制的にロックし続けます。
* **自動クリーンアップ**: タブを閉じると、そのタブに設定されていたカスタム名のデータは自動的に削除されます。

### 4. 洗練された設定ダッシュボード (Popup)
* 拡張機能アイコンをクリックすると（アイコンクリッククローズが無効な場合）、ガラスモフィズム（透過ガラス風）デザインの美しい設定画面が開きます。
* 設定可能な項目：
  * ダブルクリック機能の有効/無効
  * 必要とする修飾キーの選択
  * アイコンクリックで閉じる機能の有効/無効
  * 除外ドメインリスト（特定のサイトで機能を無効化する設定）
  * 現在開いているタブの名前変更・リセット

## 🛠 インストール方法

本拡張機能は開発者モードを使用して手動で読み込みます。

1. **Google Chrome** を開きます。
2. アドレスバーに `chrome://extensions/` と入力して移動します。
3. 画面右上にある**「デベロッパー モード」**のトグルスイッチを**オン**にします。
4. 画面左上に表示される**「パッケージ化されていない拡張機能を読み込む」**ボタンをクリックします。
5. 以下のフォルダを選択して読み込みます：
   `/Users/kazuki_route1127/.gemini/antigravity/scratch/double-click-close-tab`
6. 拡張機能の一覧に「**Double Click Closes Tab**」が表示されればインストール完了です！

## ⚠️ ご注意と制限事項

Google Chromeのセキュリティ制限（仕様）により、以下の動作制限があります：

1. **ブラウザ自体のタブバー（最上部のタブ自体）の上でのダブルクリックは検知できません。**
   * そのため、本拡張機能は「閲覧中のページ内の余白でのダブルクリック」をトリガーにして動作します。
2. **一部の特殊なページでは、ページ内ダブルクリックおよびタブ名変更が動作しません。**
   * Chromeウェブストア（拡張機能のダウンロードページ）
   * Chromeのシステムページ（`chrome://settings` や `chrome://extensions`、新規タブページ `chrome://newtab` など）
   * ローカルファイル表示（`file://` スキームのページ、拡張機能詳細で「ファイルの URL へのアクセスを許可」をONにすれば一部動作可能）
   * *※なお、拡張機能アイコンをクリックしてタブを閉じるアクションは、上記の特殊ページを含むすべてのタブで常に動作します。*
3. **アイコンクリッククローズ有効時の設定画面の開き方**：
   * 「アイコンクリックでタブを閉じる」を有効にすると、通常クリックでは設定画面が開かなくなります。
   * 設定を変更したい場合は、拡張機能アイコンを**右クリック**し、メニューから**「オプション」**を選択することで設定画面を開くことができます。
